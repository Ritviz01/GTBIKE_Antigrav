# GT Bikes

A sleek, minimalist web application for a premium motorcycle brand, GT Bikes. This project is built using Next.js and features modern design aesthetics, smooth scroll-triggered animations, and a dedicated e-commerce section.

## Features

- **Modern Hero Section**: Minimalist black-and-white aesthetic with strong typography.
- **Lineup Showcase**: Smooth scroll animations displaying different bike models.
- **Heritage Timeline**: A dedicated section covering the vintage history of GT Bikes.
- **E-Commerce Integrated**: A store section highlighting available products.

## Technologies Used

- Next.js 14
- React 18
- Framer Motion (for dynamic animations)
- Vanilla CSS (CSS Modules)
- Lucide React (for icons)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
